# Fine Tuning Guides

This section contains practical guides and recipes for fine-tuning Large Language Models (LLMs). These examples are primarily provided as Jupyter/Colab notebooks to help you get started quickly with experimentation and implementation.

Feel free to try these out!

## Index of Cookbooks

- **[Fine-tune Open-Source LLMs on Nebius Token Factory](Fine_tuning.ipynb)**
  - Learn how to fine-tune & deploy open models like Llama 3.1 directly from your dataset using Nebius Token Factory.
- **[Customer Support Real World Use Case Finetuning with DataLabs using Nebius Token Factory](customer-support-datalab-finetuning-tutorial.ipynb)**
  - Learn how to fine-tune & deploy a custom fine-tuned model for customer support usecase using DataLabs.
- **[Insurance Claims — Data Lab + Fine-Tuning + Gradio demo](insurance_claims_finetuning/README.md)**
  - Learn how to distill with a 70B teacher in Data Lab, LoRA fine-tune Llama 3.1 8B on Nebius Token Factory, deploy a serverless adapter, and compare base vs fine-tuned chat.
